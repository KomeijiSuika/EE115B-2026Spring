## 🎯 一句话口诀

<aside>
🔑

写 SOP → 按 1 的个数分组 → 相邻差 1 bit 合并（记 `-`、打勾）→ 没打勾的就是全部 PI → 列 PI 表，先锁 EPI、再数剩谁、补最少 PI。

</aside>

# 🔁 用期中 Problem 10 走一遍

$$
F(A,B,C,D)=\sum m(0,2,5,6,7,8,10,13,15)
$$

## 1️⃣ Step 1 — 转二进制，按「1 的个数」分组

只有相邻组（1 的个数差 1）才可能合并。

| 组（1 的个数） | minterms（二进制 ABCD） |
| --- | --- |
| 0 个 | 0 = 0000 |
| 1 个 | 2 = 0010；8 = 1000 |
| 2 个 | 5 = 0101；6 = 0110；10 = 1010 |
| 3 个 | 7 = 0111；13 = 1101 |
| 4 个 | 15 = 1111 |

## 2️⃣ Step 2 — 相邻组两两合并（第一级）

相邻组里只差 1 个 bit 的两项合并，差异位记 `-`，两个原项都打勾 ✓（依据吸收律 $XY+X\bar Y=X$）。

合并时把两项上下对齐写二进制，只有一位不同的就能合，那位记 `-`。第一级合并逐对列如下（左边原项都打勾 ✓ 表示“已被更大的项吸收”）：

| 合并对 | 二进制对齐 (ABCD) | 合并结果 |
| --- | --- | --- |
| m0, m2 | 0000 / 0010 | `00-0` |
| m0, m8 | 0000 / 1000 | `-000` |
| m2, m6 | 0010 / 0110 | `0-10` |
| m2, m10 | 0010 / 1010 | `-010` |
| m8, m10 | 1000 / 1010 | `10-0` |
| m5, m7 | 0101 / 0111 | `01-1` |
| m5, m13 | 0101 / 1101 | `-101` |
| m6, m7 | 0110 / 0111 | `011-` |
| m7, m15 | 0111 / 1111 | `-111` |
| m13, m15 | 1101 / 1111 | `11-1` |

## 3️⃣ Step 3 — 再合并到合不动 → 没打勾的就是全部 PI

第二级：`00-0`+`10-0`→ `-0-0`；`01-1`+`11-1`→ `-1-1`。合不动且没被打勾的项即 Prime Implicant：

| PI (cube) | 盖哪些 minterm | 乘积项 |
| --- | --- | --- |
| `-0-0` | 0, 2, 8, 10 | $\bar B\bar D$ |
| `-1-1` | 5, 7, 13, 15 | $BD$ |
| `0-10` | 2, 6 | $\bar A C\bar D$ |
| `011-` | 6, 7 | $\bar A BC$ |

## 4️⃣ Step 4 — 列 PI 表，先锁 EPI（列里只有一个勾）

先把 Prime Implicant 表列出来：**行 = PI，列 = 每个 minterm，√ = 该 PI 盖住该 minterm**。再看哪一列只有一个 √，那一行的 PI 就是必留的 EPI。

| PI | 0 | 2 | 5 | 6 | 7 | 8 | 10 | 13 | 15 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| $\bar B\bar D$ ★ | √ | √ |  |  |  | √ | √ |  |  |
| $BD$ ★ |  |  | √ |  | √ |  |  | √ | √ |
| $\bar A C\bar D$ |  | √ |  | √ |  |  |  |  |  |
| $\bar A BC$ |  |  |  | √ | √ |  |  |  |  |

看列判 EPI（★ = essential）：

- m0 / m8 / m10 列只有 $\bar B\bar D$ 一个 √ → $\bar B\bar D$ 是 EPI。
- m5 / m13 / m15 列只有 $BD$ 一个 √ → $BD$ 是 EPI。
- m6 列有两个 √（$\bar A C\bar D$ 和 $\bar A BC$）→ 都不是必留，留到 Step 5 二选一。

## 5️⃣ Step 5 — 数「还剩谁没盖」，补最少 PI

锁完两个 EPI 后，只剩 **m6** 没盖；m6 能被 $\bar A C\bar D$ 或 $\bar A BC$ 覆盖 → 二选一补一个即可：

$$
F=\bar B\bar D+BD+\bar A C\bar D
$$

<aside>
⚠️

**期中失分点：** Step 5 只剩 m6，却把 $\bar A C\bar D$ 和 $\bar A BC$ 两个等价 PI 都写上 → 多了一个冗余项，没达到 minimal SOP。**只差一个 minterm 就只补一个 PI。**

</aside>
