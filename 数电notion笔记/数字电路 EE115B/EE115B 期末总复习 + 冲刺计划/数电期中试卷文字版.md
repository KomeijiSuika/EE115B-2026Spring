## Instructions

- Solutions for comprehensive problems should be hand-written clearly in English, with all necessary derivation steps.
- Unless otherwise specified, logic simplifications should reach the minimal Sum of Products (SOP) form.
- Given `F(·)`, minterms/maxterms are defined with variable order. For example, `F(A, B, C)` has `A` (high), `B` (middle), `C` (low), where each index corresponds to the binary assignment in that order.
- The exam is out of 100 points total.

## I. Choice Problem [16 pts]

Select the **ONE** best answer for each question. (2 pts each)

1. Which of the following logic gates is considered a universal gate?
    - A. AND gate
    - B. OR gate
    - C. NOR gate
    - D. XOR gate
2. According to DeMorgan's Theorem, the Boolean expression `\overline{A + B}` is equivalent to:
    - A. `\overline{A} + \overline{B}`
    - B. `\overline{A} · \overline{B}`
    - C. `A · B`
    - D. `A + B`
3. What is the representable decimal range of an `n`-bit signed two's complement integer?
    - A. `0` to `2^n - 1`
    - B. `-2^(n-1)` to `2^(n-1) - 1`
    - C. `-2^(n-1) + 1` to `2^(n-1) - 1`
    - D. `-2^n` to `2^n - 1`
4. In a Karnaugh map, a prime implicant that covers at least one minterm not covered by any other prime implicant is called:
    - A. A static hazard
    - B. A don't-care condition
    - C. An essential prime implicant
    - D. A consensus term
5. A static-0 hazard occurs when a combinational circuit output is supposed to remain at logic `0`, but due to unequal path delays, it temporarily:
    - A. Goes to a High-Z (high impedance) state
    - B. Transitions to logic `1` and back to `0`
    - C. Oscillates infinitely between `0` and `1`
    - D. Drops to a negative voltage level
6. A multiplexer with 4 select lines has how many data inputs?
    - A. 4
    - B. 8
    - C. 16
    - D. 32
7. Which of the following binary codes is self-complementing?
    - A. Excess-3 code
    - B. Gray code
    - C. 8421 BCD code
    - D. ASCII code
8. In a 4-variable Karnaugh map, grouping 8 adjacent ones eliminates how many variables?
    - A. 1 variable
    - B. 2 variables
    - C. 3 variables
    - D. 4 variables

2. Construct the prime implicant chart, write down all essential prime implicants identified, and the final minimal SOP expression for `F`.
- ❓ Q&A — 为什么最后一题（Problem 10 / Quine–McCluskey）被扣分？（恋恋 2026-06-18 提问）
    
    **我写的最终答案（4 项）：** $F = \overline{B}\,\overline{D} + \overline{A}\,C\,\overline{D} + \overline{A}BC + BD$
    
    **正确最小 SOP（3 项）：**
    
    $$
    F = \overline{B}\,\overline{D} + BD + \overline{A}BC
    $$
    
    （第三项换成 $\overline{A}\,C\,\overline{D}$ 也对，两者等价，都只为盖住 m6。）
    
    **错因 —— 冗余项（redundant term）：** 两个 essential PI（$\overline{B}\,\overline{D}$ 盖 0/2/8/10，$BD$ 盖 5/7/13/15）已经盖住除 m6 外的全部 minterm；剩下的 m6 只需再补**一个** PI。$\overline{A}\,C\,\overline{D}$ 与 $\overline{A}BC$ 都能盖 m6，**二选一即可**。我两个都写上 → 多一个冗余项 → 没达到题目要求的 minimal SOP，被扣分。
    
    **Prime implicant chart（√ = 覆盖该 minterm）：**
    
    | PI | 0 | 2 | 5 | 6 | 7 | 8 | 10 | 13 | 15 |
    | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
    | B'D' ★ | √ | √ |  |  |  | √ | √ |  |  |
    | BD ★ |  |  | √ |  | √ |  |  | √ | √ |
    | A'CD' |  | √ |  | √ |  |  |  |  |  |
    | A'BC |  |  |  | √ | √ |  |  |  |  |
    
    ★ = essential PI（B'D' 是唯一盖 0/8/10 的项，BD 是唯一盖 5/13/15 的项）；m6 由 A'CD' 或 A'BC 二选一补上即可。
    
    **可吸取的经验：**
    
    - 选完 essential PI 后，务必回到 chart 数“还剩哪些 minterm 没被盖”——这次只剩 m6。
    - 只差一个 minterm 时只补**一个** PI；若有多个等价 PI（literal 数相同），任选其一，别都写。
    - 题目写 minimal / simplest SOP 时，“逻辑正确”≠ 满分，必须项数 / 字母数最少。
    - 交卷前自检口诀：试着删掉任意一项，若函数仍然完整覆盖，则该项冗余 → 删掉。
