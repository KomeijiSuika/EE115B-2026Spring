# EE115B 2026 Spring Digital Circuits

This repository contains notes, homework materials, review documents, and supporting learning resources for EE115B Digital Circuits.

## Contents

- `数电notion笔记/`: exported Notion notes, with Notion random page IDs removed from filenames and internal links.
- `homework1/`: Homework 1 materials and a small truth-table helper script.
- `期中/`: midterm review notes and selected practice/reference PDFs.
- `learning/`: supplementary explanations, figures, and slide previews.

## Privacy Cleanup

Personal identifiers in the tracked text files have been scrubbed where found. Student names and IDs in generated/submission-style files were normalized to placeholder values such as `古明地雷` and `114514`.

Course instructor information is kept. TA-specific personal information was generalized to course-support wording in the notes where it appeared in text.

## Ignored Files

The repository ignores local/editor state, temporary folders, build outputs, LaTeX intermediates, HDL simulation artifacts, generated netlists, and archive bundles. See `.gitignore` for the exact rules.

## Project Directory

`project/` is intentionally not tracked by this parent repository right now because it is an existing nested Git repository with its own history. To publish it safely, first decide whether to keep it as a separate repository, rewrite/remove its old history, or import only the cleaned current files as ordinary files.

## Notes

The materials are primarily for personal study and review. Some PDFs are original course/reference materials and may still contain course-provided metadata or TA wording inside the PDF content.
